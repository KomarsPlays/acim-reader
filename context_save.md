# 📁 Контекст проекта "Курс Чудес" (ACIM Reader)

**Дата сохранения:** 17.03.2026

## 🎯 Суть проекта
Веб-приложение (PWA) для чтения книги "Курс Чудес" (уроки 1-365).  
Реализовано на чистом HTML/CSS/JS. Доступно трём пользователям по одной ссылке, у каждого свой прогресс.

## 🗂️ Структура файлов (`Work/acim-reader/`)
- `lessons.json` — 405 уроков из PDF
- `index.html` — точка входа (PWA-мета, Service Worker)
- `style.css` — книжный дизайн, Bottom Sheet для мобильных
- `app.js` — логика: оглавление, чтение, навигация, фразы дня, selectionchange
- `sw.js` — Service Worker (кэш v4, офлайн-режим)
- `manifest.json` — PWA-манифест (иконки, standalone, portrait)
- `icons/` — иконки 192x192 и 512x512 (maskable)
- `.gitignore` — исключает node_modules, parse_pdf.js, build_json.js
- `build_json.js`, `parse_pdf.js` — скрипты парсинга (не в Git, не нужны на сервере)

## 💡 Особенности текущей версии
1. **PWA:** Устанавливается на рабочий стол телефона, работает офлайн.
2. **Фраза дня:** Выделение текста вызывает Bottom Sheet снизу экрана (на мобильных).  
   На десктопе — popup над выделением.
3. **Стили фраз:** Шрифт 0.9rem, без курсива, с сохранением переносов строк (`white-space: pre-line`).
4. **Производительность:** Используется `selectionchange` с дебаунсом 400мс и ранним выходом (сравнение `lastSelectedText`).
5. **Кэш:** При любых изменениях — повышать версию в `sw.js` (`CACHE_NAME`), иначе телефоны не обновятся!
