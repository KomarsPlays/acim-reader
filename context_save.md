# 📁 Контекст проекта "Курс Чудес" (ACIM Reader)

**Дата сохранения:** 04.03.2026 (или позднее, в зависимости от сессии)
**Статус:** Базовая версия готова (v1.0), парсинг PDF завершён, приложение функционирует локально.

## 🎯 Суть проекта
Веб-приложение (PWA-подобное) для чтения книги "Курс Чудес" (уроки 1-365). 
Реализовано на чистом HTML/CSS/JS без фреймворков для максимальной скорости и автономности.

## 🗂️ Структура файлов (папка `Work/acim-reader/`)
- `lessons.json` — база из 405 уроков, извлечённых из PDF.
- `index.html` — единая точка входа. Содержит структуру экрана оглавления и чтения.
- `style.css` — книжный, тёплый дизайн, отзывчивая вёрстка (цвета: `--bg-page: #FAF7F2`, `--text-main: #2C2825`, акцент `--accent: #D4A373`).
- `app.js` — бизнес-логика:
  - Рендеринг списка уроков и текста.
  - Сохранение/Восстановление позиции чтения (`localStorage: acim_scroll`).
  - Функция "Фраза дня" (сохранение выделенного текста через popup в `localStorage: acim_phrases`).
- `parse_pdf.js` / `build_json.js` — скрипты Node.js для извлечения текста из `../../z_Input/Книги/Курс Чудес.pdf`. В `build_json.js` реализована продвинутая логика сохранения абзацев (маркированные списки `1.`) и четверостиший (отступ >8 пробелов).

## 💡 Особенности реализации
1. **Фраза дня:** Пользователь выделяет текст. Всплывает диалог "Сделать фразой дня?". При "Да", фраза закрепляется в сворачиваемой верхней панели (`#phrases-bar`).
2. **Форматирование стихов:** Класс `.lesson-verse`. Выделяет стихи курсивом, сохраняет переносы строк, добавляет отступ и левую бежевую рамку.
3. **Запуск:** Обязателен запуск через локальный сервер (например, Python `http.server 8080` или `npx http-server`), иначе браузер заблокирует `fetch('lessons.json')` из-за CORS политик для `file://`.
